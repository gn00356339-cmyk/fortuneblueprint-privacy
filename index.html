藍圖問命 FortuneBlueprint · 隱私政策

最後更新:2026 年 6 月 7 日

1. 我們不蒐集個人資料

App 不蒐集任何可識別你個人身分的資料。姓名、生辰、出生地等命理輸入僅本機計算、不儲存、不上傳。

2. 命理排盤如何運作

排盤計算(紫微 12 宮、八字四柱、姓名五格、星座本命盤)走我們的 Cloudflare Workers 中轉、純算法、不打 LLM、不留 log。中轉伺服器只回排盤 JSON、不保存任何輸入。

3. LLM 解讀

付費解鎖後的 LLM 解讀文字、由 Google Gemini API 即時生成、透過加密 HTTPS、不關聯任何使用者身分。同一輸入會走 SHA256 雜湊 cache 30 天節省 API 成本、無法反推姓名 / 生辰。

4. 內購 (IAP)

Pro 解讀 NT$30、Premium 綜合論命 NT$50、皆為一次性買斷、不訂閱。購買由 Apple App Store 處理、我們不接觸付款資訊。

5. 第三方服務

  • Apple StoreKit(內購)
  • Cloudflare Workers(排盤算法中轉)
  • Google Gemini API(LLM 解讀文字生成)

6. 兒童使用

本 App 適用 12+、不刻意蒐集 12 歲以下兒童的資料。

7. 免責聲明

本 App 提供之所有命理內容,純屬娛樂、僅供參考,不構成任何專業、醫療、法律、財務或投資建議。命運由你的選擇與行動決定、不由命盤決定。

FortuneBlueprint · Privacy Policy

Last updated: June 7, 2026

1. We Don't Collect Personal Data

The App does not collect any personally identifiable information. Name, birth date, birthplace inputs are computed on-device or via a stateless relay, never stored.

2. How Astrolabe Calculation Works

Calculations (Ziwei 12 palaces, Bazi 4 pillars, Name 5-grid, Western natal chart) run on our Cloudflare Workers relay using deterministic algorithms — no LLM, no logging, no storage.

3. LLM Interpretation

Paid LLM interpretations are generated by Google Gemini API in real time via encrypted HTTPS, not tied to any user identity. Identical inputs use a SHA256 cache for 30 days; names and birthdays cannot be reverse-engineered.

4. In-App Purchases

Pro Interpretation (NT$30) and Premium Synthesis (NT$50) are one-time non-consumable purchases. Apple handles payments; we never see payment info.

5. Third-Party Services

  • Apple StoreKit
  • Cloudflare Workers (algorithm relay)
  • Google Gemini API (LLM text generation)

6. Disclaimer

All content is for entertainment and reference only. Not professional, medical, legal, financial, or investment advice. Your destiny is shaped by your choices and actions, not by a chart.
